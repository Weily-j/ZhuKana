## ADDED Requirements

### Requirement: 變音鍵循環切換
`變音` 鍵 SHALL 對組字緩衝區最後一個假名進行循環轉換：清音 → 濁音(゛) → 半濁音(゜，僅は行適用) → 小字 → 清音。

#### Scenario: か → が 濁音轉換
- **WHEN** 使用者輸入 か 後按下 變音 鍵
- **THEN** 組字緩衝區的 か 變為 が

#### Scenario: は → ば → ぱ 循環
- **WHEN** 使用者輸入 は 後連續按下 變音 鍵
- **THEN** 第一次按：は → ば；第二次按：ば → ぱ；第三次按：ぱ → 小字ぁ；第四次按：ぁ → は

#### Scenario: か 行無半濁音，跳過 ゜
- **WHEN** 使用者輸入 か 後連續按下 變音 鍵
- **THEN** 第一次：か → が；第二次：が → 小字ぁ；第三次：ぁ → か（跳過半濁音步驟）

#### Scenario: つ → っ 小字轉換
- **WHEN** 使用者輸入 つ 後按下 變音 鍵兩次（か行邏輯：清音→濁づ→小っ）
- **THEN** 最終 つ 轉換為 っ

### Requirement: 片假名模式下同樣適用
片假名模式下，`變音` 鍵 SHALL 對片假名進行相同的循環轉換。

#### Scenario: カ → ガ 片假名濁音轉換
- **WHEN** 使用者在片假名模式下輸入 カ 後按下 變音 鍵
- **THEN** カ 變為 ガ

### Requirement: 濁音注音對應顯示
在撥動展開顯示濁音字時，SHALL 顯示對應的注音組合。

#### Scenario: じ 顯示 ㄐㄧ
- **WHEN** 使用者輸入 じ（ざ行 → 變音 → い段）
- **THEN** 注音顯示 ㄐㄧ（非 ㄗㄧ），標示「合流」

#### Scenario: ぢ・づ 顯示同音合流說明
- **WHEN** 使用者輸入 ぢ 或 づ
- **THEN** 注音分別顯示 ㄐㄧ（＝じ）和 ㄗㄨ（＝ず），標示「合流」
